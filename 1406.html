문제 문제 링크

한 줄로 된 간단한 에디터를 구현하려고 한다. 이 편집기는 영어 소문자만을 기록할 수 있는 편집기로, 최대 600,000글자까지 입력할 수 있다.

이 편집기에는 '커서'라는 것이 있는데, 커서는 문장의 맨 앞(첫 번째 문자의 왼쪽), 문장의 맨 뒤(마지막 문자의 오른쪽), 또는 문장 중간 임의의 곳(모든 연속된 두 문자 사이)에 위치할 수 있다. 즉 길이가 L인 문자열이 현재 편집기에 입력되어 있으면, 커서가 위치할 수 있는 곳은 L+1가지 경우가 있다.

이 편집기가 지원하는 명령어는 다음과 같다:

  • L: 커서를 왼쪽으로 한 칸 옮김 (커서가 문장의 맨 앞이면 무시됨)
  • D: 커서를 오른쪽으로 한 칸 옮김 (커서가 문장의 맨 뒤이면 무시됨)
  • B: 커서 왼쪽에 있는 문자를 삭제함 (커서가 문장의 맨 앞이면 무시됨)
  • P $: $라는 문자를 커서 왼쪽에 추가함

초기에 편집기에 입력되어 있는 문자열이 주어지고, 그 이후 입력한 명령어가 차례로 주어졌을 때, 모든 명령어를 수행하고 난 후 편집기에 입력되어 있는 문자열을 구하는 프로그램을 작성하시오.

입력

첫째 줄에는 초기에 편집기에 입력되어 있는 문자열이 주어진다. 이 문자열은 길이가 N이고, 영어 소문자로만 이루어져 있으며, 길이는 100,000을 넘지 않는다. 둘째 줄에는 입력할 명령어의 개수를 나타내는 정수 M(1 ≤ M ≤ 500,000)이 주어진다. 셋째 줄부터 M개의 줄에 걸쳐 입력할 명령어가 순서대로 주어진다. 명령어는 위의 네 가지 중 하나의 형태로만 주어진다.

예제 입력 1

abcd
3
P x
L
P y

출력

첫째 줄에 모든 명령어를 수행하고 난 후 편집기에 입력되어 있는 문자열을 출력한다.

예제 출력 1

abcdyx

풀이

문제 해결 방법

이 문제는 스택을 사용하여 효율적으로 해결할 수 있습니다. 다음과 같은 방법으로 구현합니다:

  1. 왼쪽 스택과 오른쪽 스택을 사용하여 커서의 위치를 관리합니다.
  2. 커서는 두 스택 사이에 위치합니다.
  3. L 명령어: 왼쪽 스택의 top을 오른쪽 스택으로 이동
  4. D 명령어: 오른쪽 스택의 top을 왼쪽 스택으로 이동
  5. B 명령어: 왼쪽 스택의 top을 제거
  6. P 명령어: 왼쪽 스택에 문자 추가

시간 복잡도는 O(N+M)입니다.

코드

#include

using namespace std;

int main() {
    string initial;
    cin >> initial;
    
    stack left, right;
    
    // 초기 문자열을 왼쪽 스택에 넣기
    for (char c : initial) {
        left.push(c);
    }
    
    int M;
    cin >> M;
    
    while (M--) {
        char cmd;
        cin >> cmd;
        
        if (cmd == 'L') {
            if (!left.empty()) {
                right.push(left.top());
                left.pop();
            }
        }
        else if (cmd == 'D') {
            if (!right.empty()) {
                left.push(right.top());
                right.pop();
            }
        }
        else if (cmd == 'B') {
            if (!left.empty()) {
                left.pop();
            }
        }
        else if (cmd == 'P') {
            char c;
            cin >> c;
            left.push(c);
        }
    }
    
    // 왼쪽 스택의 내용을 오른쪽 스택으로 이동
    while (!left.empty()) {
        right.push(left.top());
        left.pop();
    }
    
    // 오른쪽 스택의 내용을 출력
    while (!right.empty()) {
        cout << right.top();
        right.pop();
    }
    
    return 0;
}
