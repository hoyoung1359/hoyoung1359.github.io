문제 문제 링크

고스택은 스택을 이용한 간단한 프로그래밍 언어이다. 이 언어는 10가지의 명령어를 가지고 있으며, 스택을 이용하여 연산을 수행한다.

명령어는 다음과 같다:

  • NUM X: X를 스택의 가장 위에 저장한다. (0 ≤ X ≤ 10^9)
  • POP: 스택 가장 위의 숫자를 제거한다.
  • INV: 첫 번째 수의 부호를 바꾼다.
  • DUP: 첫 번째 숫자를 하나 더 스택의 가장 위에 저장한다.
  • SWP: 첫 번째 숫자와 두 번째 숫자의 위치를 서로 바꾼다.
  • ADD: 첫 번째 숫자와 두 번째 숫자를 더한다.
  • SUB: 첫 번째 숫자에서 두 번째 숫자를 뺀다.
  • MUL: 첫 번째 숫자와 두 번째 숫자를 곱한다.
  • DIV: 첫 번째 숫자를 두 번째 숫자로 나눈다.
  • MOD: 첫 번째 숫자를 두 번째 숫자로 나눈 나머지를 구한다.

입력

입력은 여러 개의 프로그램으로 이루어져 있다. 각 프로그램은 명령어로 이루어져 있으며, 명령어는 한 줄에 하나씩 주어진다. 프로그램의 끝은 QUIT으로 나타내며, QUIT 명령어는 프로그램에 포함되지 않는다.

각 프로그램의 첫 번째 줄에는 프로그램을 실행할 때 스택에 넣을 숫자의 개수 N이 주어진다. (1 ≤ N ≤ 10,000)

다음 N개의 줄에는 프로그램을 실행할 때 스택에 넣을 숫자가 주어진다. (0 ≤ 숫자 ≤ 10^9)

출력

풀이

문제 해결 방법

이 문제는 스택을 이용한 간단한 프로그래밍 언어를 구현하는 문제입니다. 주요 해결 단계는 다음과 같습니다:

  1. 명령어 파싱:
    • 각 명령어를 문자열로 입력받아 처리
    • NUM 명령어의 경우 추가로 숫자 파라미터 처리
  2. 스택 연산 구현:
    • 각 명령어에 따른 스택 연산 구현
    • 오버플로우/언더플로우 처리
    • 0으로 나누기 등의 예외 처리
  3. 프로그램 실행:
    • 각 입력 숫자에 대해 프로그램 실행
    • 오류 발생 시 즉시 중단하고 ERROR 출력

이 알고리즘의 시간 복잡도는 O(NM)입니다. (N: 명령어 개수, M: 입력 숫자 개수)

using namespace std;

struct Command {
    string cmd;
    int num;
};

vector program;
stack st;

bool execute() {
    for (const auto& cmd : program) {
        if (cmd.cmd == "NUM") {
            st.push(cmd.num);
        }
        else if (cmd.cmd == "POP") {
            if (st.empty()) return false;
            st.pop();
        }
        else if (cmd.cmd == "INV") {
            if (st.empty()) return false;
            long long x = st.top(); st.pop();
            st.push(-x);
        }
        else if (cmd.cmd == "DUP") {
            if (st.empty()) return false;
            st.push(st.top());
        }
        else if (cmd.cmd == "SWP") {
            if (st.size() < 2) return false;
            long long a = st.top(); st.pop();
            long long b = st.top(); st.pop();
            st.push(a);
            st.push(b);
        }
        else if (cmd.cmd == "ADD") {
            if (st.size() < 2) return false;
            long long a = st.top(); st.pop();
            long long b = st.top(); st.pop();
            if (abs(a + b) > 1e9) return false;
            st.push(a + b);
        }
        else if (cmd.cmd == "SUB") {
            if (st.size() < 2) return false;
            long long a = st.top(); st.pop();
            long long b = st.top(); st.pop();
            if (abs(b - a) > 1e9) return false;
            st.push(b - a);
        }
        else if (cmd.cmd == "MUL") {
            if (st.size() < 2) return false;
            long long a = st.top(); st.pop();
            long long b = st.top(); st.pop();
            if (abs(a * b) > 1e9) return false;
            st.push(a * b);
        }
        else if (cmd.cmd == "DIV") {
            if (st.size() < 2) return false;
            long long a = st.top(); st.pop();
            long long b = st.top(); st.pop();
            if (a == 0) return false;
            st.push(b / a);
        }
        else if (cmd.cmd == "MOD") {
            if (st.size() < 2) return false;
            long long a = st.top(); st.pop();
            long long b = st.top(); st.pop();
            if (a == 0) return false;
            st.push(b % a);
        }
    }
    return true;
}

int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);

    while (true) {
        program.clear();
        string s;
        while (cin >> s) {
            if (s == "QUIT") break;
            if (s == "END") break;
            Command cmd;
            cmd.cmd = s;
            if (s == "NUM") {
                int x;
                cin >> x;
                cmd.num = x;
            }
            program.push_back(cmd);
        }
        if (s == "QUIT") break;

        int n;
        cin >> n;
        while (n--) {
            int x;
            cin >> x;
            st = stack();
            st.push(x);
            if (execute() && st.size() == 1) {
                cout << st.top() << '\n';
            }
            else {
                cout << "ERROR\n";
            }
        }
        cout << '\n';
    }
    return 0;
}
