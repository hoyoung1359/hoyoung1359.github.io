문제 문제 링크

스택 (stack)은 기본적인 자료구조 중 하나로, 컴퓨터 프로그램을 작성할 때 자주 이용되는 개념이다. 스택은 자료를 넣는 (push) 입구와 자료를 뽑는 (pop) 입구가 같아 제일 나중에 들어간 자료가 제일 먼저 나오는 (LIFO, Last in First out) 특성을 가지고 있다.

1부터 n까지의 수를 스택에 넣었다가 뽑아 늘어놓음으로써, 하나의 수열을 만들 수 있다. 이때, 스택에 push하는 순서는 반드시 오름차순을 지키도록 한다고 하자. 임의의 수열이 주어졌을 때 스택을 이용해 그 수열을 만들 수 있는지 없는지, 있다면 어떤 순서로 push와 pop 연산을 수행해야 하는지를 알아낼 수 있다. 이를 계산하는 프로그램을 작성하라.

입력

첫째 줄에 n (1 ≤ n ≤ 100,000)이 주어진다. 둘째 줄부터 n개의 줄에 걸쳐 수열을 이루는 1이상 n이하의 정수가 하나씩 순서대로 주어진다. 물론 같은 정수가 두 번 나오는 일은 없다.

문제 해결 방법

이 문제는 스택을 사용하여 주어진 수열을 만들 수 있는지 확인하고, 가능하다면 push와 pop 연산의 순서를 출력하는 문제입니다. 다음과 같은 방법으로 구현합니다:

  1. 현재 스택에 넣을 수 있는 숫자를 추적하는 변수(current)를 사용합니다.
  2. 수열의 각 숫자에 대해:
    • 현재 숫자가 current보다 크면, current부터 해당 숫자까지 push합니다.
    • 스택의 top이 현재 숫자와 같으면 pop합니다.
    • 스택의 top이 현재 숫자와 다르면 수열을 만들 수 없습니다.

시간 복잡도는 O(N)입니다.

코드

#include 
#include 
#include 
#include 

using namespace std;

int main() {
    int n;
    cin >> n;
    
    vector target(n);
    for (int i = 0; i < n; i++) {
        cin >> target[i];
    }
    
    stack s;
    vector result;
    int current = 1;
    
    for (int i = 0; i < n; i++) {
        while (current <= target[i]) {
            s.push(current);
            result.push_back("+");
            current++;
        }
        
        if (s.top() == target[i]) {
            s.pop();
            result.push_back("-");
        } else {
            cout << "NO" << endl;
            return 0;
        }
    }
    
    for (const string& op : result) {
        cout << op << "\n";
    }
    
    return 0;
}
