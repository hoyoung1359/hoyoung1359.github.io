1. 동기화 기본 개념

면접 대비: 동기화 문제와 해결 방법은 OS 면접의 핵심 주제입니다.

1.1 경쟁 상태

  • 정의: 여러 프로세스가 동시에 공유 데이터에 접근할 때 발생하는 문제
  • 예시
    •   공유 변수 동시 수정
    •   공유 자원 동시 접근

1.2 임계 영역

  • 정의: 공유 데이터에 접근하는 코드 영역
  • 해결 조건
    •   상호 배제
    •   진행
    •   한정 대기

2. 동기화 도구

2.1 뮤텍스 락

면접 대비: 뮤텍스와 세마포어의 차이점을 설명할 수 있어야 합니다.

  • 원리: acquire()와 release() 연산으로 임계 영역 보호
  • 특징
    •   단순한 동기화 도구
    •   바쁜 대기 발생 가능

2.2 세마포어

  • 원리: 정수 변수와 wait(), signal() 연산으로 동기화
  • 유형
    •   카운팅 세마포어
    •   이진 세마포어

2.3 모니터

참고: 모니터는 고수준 동기화 도구로, Java의 synchronized 키워드가 대표적입니다.

  • 특징
    •   조건 변수 사용
    •   상호 배제 자동 보장

3.2 교착 상태 처리

    •   은행원 알고리즘
  3. 탐지 및 복구
