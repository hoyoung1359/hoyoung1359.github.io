문제 문제 링크

문자열 `s`가 주어졌을 때, 괄호가 올바르게 닫혔는지 확인하세요. 문자열은 다음 문자들로만 구성됩니다: '(', ')', '{', '}', '[', ']'.

입력

문자열 `s`가 주어집니다. 문자열의 길이는 1 이상 10,000 이하입니다.

예제 입력 1

s = "()"

출력

괄호가 올바르게 닫혔으면 `true`를, 그렇지 않으면 `false`를 반환합니다.

예제 출력 1

true

풀이

문제 해결 방법

이 문제는 다음과 같은 방법으로 해결할 수 있습니다:

  1. 스택을 사용한 방법:
    • 여는 괄호를 만나면 스택에 push
    • 닫는 괄호를 만나면 스택의 top과 짝이 맞는지 확인
    • 짝이 맞으면 pop, 아니면 false 반환
    • 마지막에 스택이 비어있어야 true

시간 복잡도는 O(N)이고, 공간 복잡도는 O(N)입니다.

코드

class Solution {
public:
    bool isValid(string s) {
        stack st;
        for (char c : s) {
            if (c == '(' || c == '{' || c == '[') {
                st.push(c);
            } else {
                if (st.empty()) return false;
                char top = st.top();
                if ((c == ')' && top != '(') ||
                    (c == '}' && top != '{') ||
                    (c == ']' && top != '[')) {
                    return false;
                }
                st.pop();
            }
        }
        return st.empty();
    }
};
